# doctest for C
> source code: [test/doctest.py](test/doctest.py)

## Usage


Taken [src/aparsearg.h](src/aparsearg.h) as an example:

```shell
cd test
python doctest.py aparsearg
```

More compile unit can be appended,

e.g. `python doctest.py streams strutils`
